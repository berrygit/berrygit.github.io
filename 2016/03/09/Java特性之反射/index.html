类对象

每个类在加载后，在虚拟机的堆内存中都对应一个Class类型的对象。该类对象保持着这个类的各种有用信息，通过获得该类对象的引用我们可以在运行时获得该类的类型信息。

获得类对象引用的方式：

  1. Class.forName("Gum");，静态方法，初始化类。
  2. object.getClass();，Object类中的方法。
  3. Test.class，类常量，不初始化类。

类对象的几个方法：
newInstance()，调用无参构造器，生成实例，返回的是Object需要自己cast。
